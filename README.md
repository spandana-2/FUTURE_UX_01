# FUTURE_UX_01: Local Clinic Website Redesign

## Task Overview
Redesign a local clinic website focusing on **Lead Generation** and **Trust Building**.

## 🛠 Tools Used
- **Visuals:** Figma / Adobe XD
- **Prototyping:** HTML5 & Tailwind CSS
- **Documentation:** GitHub

## 🎯 Design Decisions & Rationale
1. **Lead Generation:** Placed a sticky CTA in the header and a prominent form in the middle of the page to ensure zero-friction booking.
2. **Trust Elements:** Used "Medical Blue" (#2563eb) to evoke professionalism and reliability. 
3. **Mobile First:** The layout uses a responsive grid that collapses into a single-column view for mobile users, making buttons easy to tap.
4. **Fitts's Law:** Navigation and "Call Now" buttons are oversized for better accessibility on small screens.

## 📁 Repository Structure
- `/assets`: Contains high-fidelity UI screenshots.
- `index.html`: The functional UI prototype code.
- `Rationale.pdf`: Detailed UX decision document.

## 🚀 How to View
Open `index.html` in any modern web browser to view the responsive UI.
